## Spring

### Spring介绍

官网：[Spring | Home](https://spring.io/)

#### 为什么要学

* Spring技术是JavaEE开发必备技能，企业开发技术选型命中率>90%

* 专业角度：
  * **简化开发**，降低企业级开发的复杂性
  * **框架整合**，高效整合其他技术，提高企业级应用开发与运行效率

#### 学什么

* 简化开发
  * IoC
  * AOP
    * 事务处理

* 框架整合
  * MyBatis
  * MyBatis-Plus
  * Struts
  * Struts2
  * Hibernate
  * ......
